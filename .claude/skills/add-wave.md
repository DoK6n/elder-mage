# Add Wave

새로운 몬스터 스폰 웨이브를 추가합니다.

## Usage
```
/add-wave [몬스터타입] [시작시간] [종료시간] [스폰수] [스폰주기]
```

## Parameters
- **몬스터타입**: EnemyType enum 값 (slime, goblin, kobold, lizardman, orc, ogre)
- **시작시간**: 웨이브 시작 시간 (초)
- **종료시간**: 웨이브 종료 시간 (초, 'none'이면 계속)
- **스폰수**: 한번에 스폰되는 몬스터 수
- **스폰주기**: 스폰 간격 (초)

## Workflow

1. **EnemySpawnSystem.ts** 수정
   - `waves` 배열에 새 웨이브 객체 추가

## Example
```
/add-wave goblin 30 120 6 1.5
```

## 웨이브 템플릿

```typescript
// EnemySpawnSystem.ts - waves 배열에 추가
{
  time: 30,                    // 시작 시간 (초)
  endTime: 120,                // 종료 시간 (undefined면 계속)
  enemyType: EnemyType.Goblin,
  count: 6,                    // 한번에 스폰 수
  spawnRate: 1.5,              // 스폰 주기 (초)
  healthMultiplier: 1,         // 체력 배율
  damageMultiplier: 1,         // 데미지 배율
},
```

## 현재 웨이브 구성
| 몬스터 | 시작 | 종료 | 스폰수 | 주기 | 체력배율 |
|--------|------|------|--------|------|----------|
| Slime | 0초 | 120초 | 5 | 2.0초 | 1.0 |
| Slime (강화) | 120초 | - | 6 | 2.5초 | 1.5 |
| Goblin | 60초 | - | 4 | 2.0초 | 1.0 |
| Kobold | 120초 | - | 5 | 1.8초 | 1.0 |
| Lizardman | 180초 | - | 3 | 2.5초 | 1.0 |
| Orc | 240초 | - | 2 | 3.0초 | 1.0 |
| Ogre (보스) | 300초 | - | 1 | 45초 | 1.0 |

## 밸런스 팁
- **초반 (0-60초)**: 약한 몬스터, 적은 수
- **중반 (60-180초)**: 다양한 몬스터 혼합
- **후반 (180초+)**: 강한 몬스터, 많은 수
- **보스**: 긴 스폰 주기 (30-60초)

## 동시 웨이브 시스템
- 여러 웨이브가 동시에 활성화 가능
- 각 웨이브는 독립적인 스폰 타이머 사용
- `endTime`으로 특정 시간에 웨이브 종료 가능
